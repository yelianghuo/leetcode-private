给定一个非负索引 rowIndex，返回「杨辉三角」的第 rowIndex 行。

在「杨辉三角」中，每个数是它左上方和右上方的数的和。

![图片](../images/118.gif)

示例 1:

输入: rowIndex = 3
输出: [1,3,3,1]
示例 2:

输入: rowIndex = 0
输出: [1]
示例 3:

输入: rowIndex = 1
输出: [1,1]
 

提示:

0 <= rowIndex <= 33
 

进阶：

你可以优化你的算法到 O(rowIndex) 空间复杂度吗？


思路: 利用滚动数据,每层从右到左遍历，实现o(n)的空间

res[0] 1
res[1] 1 1
res[2] 1 2 1
res[3] 1 3 3 1



```go
func getRow(rowIndex int) []int {
    if rowIndex == 0 {
        return []int{1}
    }
    if rowIndex == 1 {
        return []int{1, 1}
    }
    res := make([]int, rowIndex+1)
    res[0], res[1] = 1, 1

    for row := 2; row < rowIndex+1; row++ {
        for i := row; i > 0; i-- {
            if i == row || i == 0 {
                res[i] = 1
                continue
            }
            res[i] = res[i] + res[i-1]
        }
    }
    return res
}
```
